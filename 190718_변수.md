### 변수 및 자료형

> **dust = 60**을 해석하면?
>
> 상자를 떠올려라!!
>
> dust는 60이다 (X)
>
> dust에 60을 저장한다(O)

- 변수는 = 을 통해 할당된다.

- 자료형 확인은 type()

- 변수의 메모리 주소 확인은 id()

  ```python
  name = 'subin'
  print(name)
  
  type(name)
  
  id(name)
  ```

- 같은 값을 동시에 할당 가능

- 다른 값을 동시에 할당 가능

- 서로 값을 바꾸는 것 가능

  ```python
  x = y = 100
  print(x, y)
  
  x, y = 1, 2
  print(x, y)
  ```
